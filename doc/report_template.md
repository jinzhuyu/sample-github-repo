# Report Template

## Title

## 1. Introduction
Briefly explain the problem and why it matters.

## 2. Data
Describe the dataset, variables, source, and preprocessing steps.

## 3. Methodology
Explain the model, algorithm, or analytical workflow.

## 4. Results
Present key tables, figures, and quantitative findings.

## 5. Discussion
Interpret the results. Discuss strengths, weaknesses, and limitations.

## 6. Conclusion
Summarize what was learned and suggest future improvements.

## References
List the papers, software packages, and datasets used.
